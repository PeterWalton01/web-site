Appointment Booking Site - a PERN Full-stack Application

This page contains information about an Appointments Booking Application. The application is designed to support appointment scheduling for a set of physiotherapists. Adaption to scheduling in other environments should not be difficult. This was completed as part of the Full-Stack Development Course at CodeCademy.


  • E-Commerce Site

  • This site is a simple E-Commerce Site that allows:

    • Customer registration.
    • Customer login and logout.
    • Product browsing leading to the creation of customer orders.
    • Placement of order with payments taken against an imaginary pre-payed account.
    • The ability for a customer to view their order history.


This application consists of a server which is developed using Node, Expressjs and a PostGres database, and a client application that uses Node, React and the Redux tool-kit. The server implements a REST API. Further details can be found in the following two repositories:

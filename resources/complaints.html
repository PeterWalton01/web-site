Data Science and Machine Learning Capstone Project

This page contains information about a machine learning project concerning complains made to The New York Housing Department. This was completed as the final module in IBM Professional Certificate in Python Data Science and Machine Learning. The project was supplied with two data feeds - this first was a set of complaints raised with the housing department, the second feed was a set of characteristics of the building concerned.


  • Statement of Requirements

  • The department wishes to answer the following questions:

    1. Which type of complaint should the Department of Housing Preservation and Development of New York City focus on first?
    2. Should the Department of Housing Preservation and Development of New York City focus on any particular set of boroughs, ZIP codes, or street (where the complaints are severe) for the specific type of complaints you identified in response to Question 1?
    3. Does the Complaint Type that you identified in response to question 1 have an obvious relationship with any particular characteristic or characteristics of the houses or buildings?
    4. Can a predictive model be built for a future prediction of the possibility of complaints of the type that you have identified in response to question 1?

    The following four links lead to the analysis and conclusion for each question:

    1. Identify top complaint.
    2. Areas raising exceptional number of HEAT/HOT WATER and HEATING complaints.
    3. Analysis of feature importance.
    4. Model development and evaluation.

    Note: each analysis has a final notebook cell that contains the conclusion of the analysis.


The analysis made used of jupyter notepad. The following pyhton modules were used: pandas, numpy, matplotlib seaborn, scipy, sklearn, xgboost
